# MCP Web Types

Shared TypeScript type definitions and Zod schemas used across all `@mcp-web/*` packages.

## Installation

```bash
npm install @mcp-web/types
```

## Usage

```typescript
import type { ToolDefinition, MCPWebConfig, QueryMessage } from '@mcp-web/types';
```

## Modules

| Module | Description |
|--------|-------------|
| `tools` | Tool definition types and schemas |
| `apps` | MCP App definition types and schemas |
| `config` | Bridge and client configuration types |
| `query` | Query/response message types |
| `resources` | MCP resource types |
| `mcp` | Core MCP protocol types |
| `errors` | Error types and codes |
